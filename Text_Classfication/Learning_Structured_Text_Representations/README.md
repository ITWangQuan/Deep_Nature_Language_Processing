# Ensemble for text classification  
 
Tensorflow implementation of [Learning Structured Text Representations](https://arxiv.org/abs/1705.09207)  
Based on the excellent work of https://github.com/nlpyang/structured

**Requirements:**  

* Python 3  
* TensorFlow >= 1.4  
* gensim


**How to run:**  
  ```
  python main.py
  ```
